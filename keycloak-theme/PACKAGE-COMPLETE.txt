TROUBLESHOOTING
===============

Theme Not Showing:
  1. Check files exist: docker exec uchub-keycloak ls /opt/keycloak/themes/uc-1-pro
  2. Check permissions: docker exec uchub-keycloak ls -la /opt/keycloak/themes/uc-1-pro
  3. Clear browser cache: Ctrl+Shift+R
  4. Check logs: docker logs uchub-keycloak | grep -i theme

Logo Not Displaying:
  1. Verify file exists: ls keycloak-theme/uc-1-pro/login/resources/img/colonel-logo.png
  2. Check browser console for 404 errors
  3. Verify file format (PNG, JPG, SVG)
  4. Check file permissions

Styles Not Applied:
  1. Clear browser cache (hard reload)
  2. Check CSS loaded in DevTools Network tab
  3. Verify theme.properties has correct CSS path
  4. Check for CSS syntax errors in browser console

Container Errors:
  1. Fix permissions: docker exec -u root uchub-keycloak chown -R keycloak:keycloak /opt/keycloak/themes/uc-1-pro
  2. Restart container: docker restart uchub-keycloak
  3. Check logs: docker logs uchub-keycloak
